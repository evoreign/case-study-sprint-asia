## Run case study 1

Clone the project

```bash
  git clone https://github.com/evoreign/case-study-sprint-asia
```

Ran the script

```bash
  python main.py
```


## Run case study 2 API

Clone the project

```bash
  git clone https://github.com/evoreign/case-study-sprint-asia
```

Change dir to api

```bash
  cd api
```

Install node package

```bash
  npm install
```

Run API

```bash
  npm start
```
